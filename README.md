# PDF Flipbook - 온라인/오프라인 PDF 뷰어

## 소개

**PDF Flipbook**은 웹 기반의 현대적이고 세련된 PDF 뷰어입니다. 책장을 넘기는 듯한 자연스러운 애니메이션과 함께 PDF를 전체화면으로 감상할 수 있습니다. 특별한 설치 없이 단 하나의 HTML 파일로 완벽한 오프라인 작동을 지원합니다.

---

## 🎯 주요 기능

### 핵심 기능
- **온전한 오프라인 지원**: 인터넷 연결 없이 완벽하게 작동
- **단일 파일 배포**: index.html 하나만으로 모든 기능 사용 가능
- **완전 반응형 설계**: 모바일, 태블릿, 데스크톱 모두 최적화
- **동적 페이지 전환**: 부드러운 슬라이드 애니메이션 (400ms)
- **플로팅 메뉴**: 반투명 디자인의 직관적 컨트롤 패널

### PDF 로드 방식
- **로컬 파일**: "파일 찾아보기" 버튼으로 컴퓨터에서 PDF 선택
- **URL 입력**: 웹 주소를 입력하여 인터넷의 PDF 불러오기

### 페이지 네비게이션
| 기능 | 설명 | 아이콘 |
|------|------|--------|
| **처음** | 첫 번째 페이지로 이동 | ⏮️ |
| **이전** | 이전 페이지로 이동 | ◀️ |
| **다음** | 다음 페이지로 이동 | ▶️ |
| **끝** | 마지막 페이지로 이동 | ⏭️ |
| **1쪽보기** | 한 페이지씩 표시 | 📄 |
| **2쪽보기** | 책처럼 두 페이지 동시 표시 | 📖 |
| **화면회전** | 기기 회전 시 레이아웃 조정 안내 | 🔄 |
| **초기화** | 파일 선택 화면으로 복귀 | 🔄 |

### 추가 네비게이션
- **화면 터치/클릭**: 화면 좌측 클릭으로 이전 페이지, 우측 클릭으로 다음 페이지 이동
- **키보드 단축키**: 화살표 키로도 페이지 이동 가능
- **페이지 표시**: 현재 페이지 / 전체 페이지 수 표시 (예: "3/50")

## 📥 설치 및 사용 방법

### 1단계: 파일 다운로드
```
index.html 파일을 다운로드하세요.
```

### 2단계: 실행
**Windows:**
1. 다운로드한 `index.html` 파일을 찾습니다
2. 파일을 우클릭하고 "다른 프로그램으로 열기" → 웹 브라우저 선택
3. 또는 파일을 웹 브라우저 아이콘으로 드래그앤드롭

**macOS/Linux:**
1. `index.html` 파일을 더블클릭
2. 기본 웹 브라우저에서 자동으로 열림

### 3단계: PDF 불러오기
**파일 업로드 방식:**
1. 초기 화면에서 "파일 찾아보기" 버튼 클릭
2. 컴퓨터에서 PDF 파일 선택
3. 자동으로 플립북 뷰어로 전환

**URL 방식:**
1. "URL 입력" 필드에 PDF 웹 주소 입력 (예: https://example.com/file.pdf)
2. "불러오기" 버튼 클릭
3. PDF가 로드되어 표시됨

### 4단계: 페이지 이동
- **마우스 클릭**: 화면 좌우 클릭으로 페이지 이동
- **플로팅 메뉴**: 화면 상단의 메뉴 버튼 클릭
- **키보드**: 좌우 화살표 키 사용

## 🎨 화면 인터페이스

### 초기 화면
- PDF 파일 업로드 또는 URL 입력
- 깔끔하고 모던한 UI 디자인
- 파일 선택 후 자동 로드

### 플립북 뷰어
- **전체화면 표시**: PDF가 전체 화면을 채우도록 표시
- **플로팅 메뉴**: 화면 상단에 반투명 메뉴 고정
- **페이지 카운터**: 현재 페이지/전체 페이지 표시
- **속성 표기**: 우측 하단에 제작 정보 표시

### 반응형 디자인
| 기기 | 기본 설정 | 특징 |
|------|---------|------|
| **모바일** | 1쪽 보기 | 세로 모드 최적화, 터치 네비게이션 |
| **태블릿** | 1쪽 또는 2쪽 | 중간 크기 화면 대응 |
| **데스크톱** | 2쪽 보기 | 책장 형식 표시 |

### 특징
- **외부 의존성 없음**: 모든 라이브러리 내장
- **단일 파일**: 배포 및 관리 용이
- **최적화**: 효율적인 메모리 사용 및 빠른 로딩

## 🌐 오프라인 작동 보장

### 완벽한 오프라인 지원
이 애플리케이션은 **100% 오프라인 작동**을 보장합니다.

- ✅ 인터넷 연결 불필요
- ✅ CDN 의존성 없음
- ✅ 모든 라이브러리 내장
- ✅ 로컬 파일(`file://` 프로토콜)에서 작동
- ✅ CORS 오류 없음

### 작동 확인
```
1. 인터넷 연결 끊기
2. index.html 파일 더블클릭
3. 모든 기능 정상 작동 확인
```

## 🐛 문제 해결

### PDF가 로드되지 않는 경우
**해결책:**
1. 파일 형식이 PDF인지 확인
2. 파일이 손상되지 않았는지 확인
3. 다른 PDF 파일로 시도

### 메뉴가 작동하지 않는 경우
**해결책:**
1. 페이지가 완전히 로드될 때까지 대기
2. 브라우저 새로고침 (F5)
3. 다른 브라우저 시도

### 화면이 깨져 보이는 경우
**해결책:**
1. 브라우저 확대/축소 리셋 (Ctrl+0)
2. 전체화면 모드 종료 후 재시작
3. 브라우저 캐시 삭제

### URL로부터 PDF 로드 실패
**원인**: CORS(Cross-Origin) 정책
**해결책:**
1. 로컬 파일 업로드 방식 사용
2. CORS 문제가 없는 다른 URL 시도
